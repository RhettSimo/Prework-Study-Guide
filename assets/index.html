HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element contains the information inside the head element needed for the website to function, such as the font and webpage size (the user does not see this).
  • The header element displays the title of the webpage (ex. the title of this webpage is Prework Study Guide).
  • The header is divided by numbers. h1 represents the title, so it should only be used once.
  • The elements "ul" and "li" represent an unordered list and bullet points, respectively.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border creates a border around the content with a width of 5px and a designated color.
  • A box shadow adds a shadow around an element's frame.
  • .card selects all the above properties.
  • "link rel="stylesheet" href="./assets/style.css" links an external CSS style sheet.

Git

  • git checkout main: brings your back to the main branch.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone: copies your repository from an online location to your local computer (ex. Github.com to Github desktop).
  • git push: transmits local branch commits to your online branches
  • git pull: fetches and merges any branches from your tracking online branches
  • git add (-A): adds (all) working branch files to the main branch
  • git commit (-m): commits changes to your online repository (-m adds a message)

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A string is used to show and manipulate an order of characters.
  • A "for" loop makes a loop that loops through a block of code a certain number of times.
  • A boolean object is an object that holds a true or false value.
  • A "if" statement executes a code block if the condition is "true."
